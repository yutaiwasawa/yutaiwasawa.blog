コンテンツ要素が少ない場合でもブラウザの最下部にフッターを表示させる方法

#CSS

通常ブラウザの表示領域よりコンテンツ要素が少ないと、フッターの下に余白ができてしまいます。そんな時にFlexboxとViewport Heightを使えば、コンテンツ要素が少なくても、フッターをブラウザの最下部に表示させることができます。

表示方法

CODE

<header>HEADER</header>
<main class="content">CONTENT</main>
<footer>FOOTER</footer>

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.content {
  flex: 1 0 auto;
}

DEMO

.contentにはmarginやpaddingを一切入れていないですが、footerがブラウザの最下部にきていることがわかるかと思います。

考察

1.各要素を縦に並べる
まず、display: flexで各要素をflexコンテナボックスにし、flex-direction: columnで表示方法を縦（上から下）にします。

2.ブラウザ表示領域いっぱいに広げる
まだこの状態ではブラウザいっぱいに広がってないので、footerの下には余白があります。
ここでViewport Heightの出番です。これは、viewport（ブラウザの表示領域）を基準にした相対単位です。1vh = viewportの高さの1/100なので、bodyに対してmin-height: 100vhを設定することで全体がブラウザいっぱいに広がります。

3.余白を埋める
この状態でも見た目的には大丈夫なのですが、実際には.contentとfooterには余白があります。
なので、.contentに対してflex: 1 0 autoを指定して余白を埋めます。
またflex: 1でもいいのですが、確かIE11だと反映されなかったような…。なので一応ショートハンドでflex-shrinkとflex-basisも指定してます。

最後に

以前はpositionを指定したり、JavaScriptを使って実現していましたが、Flexboxのおかげでかなりスッキリ書けるようになりました。本当にありがたい。
ただ相変わらずIEがね…
