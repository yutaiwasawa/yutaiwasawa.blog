コンテンツ要素が少ない場合でもブラウザの最下部にフッターを表示させる方法

#CSS

通常ブラウザの表示領域よりコンテンツ要素が少ないと、フッターの下に余白ができてしまいます。
そんな時にFlexboxとViewport Heightを使えば、コンテンツ要素が少なくても、フッターをブラウザの最下部に表示させることができます。

表示方法

<header>HEADER</header>
  <main class="content">CONTENT</main>
<footer>FOOTER</footer>
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.content {
  flex: 1 0 auto;
}

DEMO

.contentにはmarginやpaddingを一切入れていないですが、footerがブラウザの最下部にきていることがわかるかと思います。

考察

1.各要素を縦に並べる
まず、display: flexで各要素をflexコンテナボックスにし、flex-direction: columnで表示方法を縦（上から下）にします。
2.ブラウザ表示領域いっぱいに広げる
この状態でも見た目的には大丈夫なのですが、実際には.contentとfooterには余白があります。なので、.contentに対してflex: 1 0 autoを指定して余白を埋めます。
またflex: 1でもいいのですが、確かIE11だと反映されなかったような…。なので一応ショートハンドでflex-shrinkとflex-basisも指定してます。

最後に

以前はpositionを指定したり、JavaScriptを使って実現していましたが、Flexboxのおかげでかなりスッキリ書けるようになりました。本当にありがたい。

ただ相変わらずIEがね…
