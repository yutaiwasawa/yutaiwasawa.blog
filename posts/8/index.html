【git】tagの使い方

#git#備忘録

リリースするときにタグをつけるので備忘録。

タグを打つ

# 指定のコミットIDにタグを打つ（注釈なし）
$ git tag [タグ名] [コミットID]

# 指定のコミットIDにタグを打つ（注釈あり）
$ git tag -a [タグ名] -m ["コメント"]

現在のブランチの最新のコミットにタグを打つ

$ git tag [タグ名]

タグの一覧を見る

ローカルのタグ一覧を見る

$ git tag

リモートのタグ一覧を見る

$ git ls-remote --tags

リモートに反映させる

指定のタグをリモートに反映させる

$ git push origin [タグ名]

全てのタグをリモートに反映させる

$ git push origin --tags

タグを削除

ローカルのタグを削除

$ git tag -d [タグ名]

リモートのタグを削除

$ git push origin :refs/tags/[タグ名]
