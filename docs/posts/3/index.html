高さが固定ではない要素を天地左右の中央に配置する方法

#CSS

前回に続けてFlexbox活用第2弾です。

今回はこれまで高さが固定ではない要素を、天地左右の中央に配置することは難しかったのですが、Flexboxを使うことによってスッキリ簡単に実現可能になりました。

表示方法

<div class="container">
  <div class="content">高さが固定ではないコンテンツ</div>
</div>
.container {
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
}

DEMO

たったCSSを4行指定しただけで、天地左右の中央に要素を持ってこられます。

考察

まず親要素である.containerに対して高さをを指定します。今回はheight: 100vhでブラウザいっぱいに広げています。更にdisplay: flexを指定して.contentをFlexコンテナボックスにし、align-items: centerで天地中央に、justify-content: centerで左右中央にします。

最後に

高さを指定しないで天地左右の中央に持ってこられることによって、ブログのタイトルなど長さが決まっていない動的な要素を入れられるようになります。これも本当にありがたい。
