アートアクアリウム展に行ってきた

#アートアクアリウム展#イベント

今年は記念すべき10周年ということで、東京・日本橋のアートアクアリウム展に行ってきた。
約8000匹もの金魚がいるらしい。

撮影した写真

円柱の水槽。下にビー玉が敷き詰めてあるのがまたオシャレ。

目デカっ！

この藻？もいい具合に映える。

手毬をモチーフにした水槽。

ニモもいた。『ファインディング・ドリー』とコラボ。

中は金魚ではないけどキレイ。

蓮がモチーフ。自然と調和していて水族館にありそう。

蓮がモチーフ。ピンクver.。

すごい数の金魚。

グラスに入る金魚。

白い金魚。

口紅金魚。なんか優雅。

写真だとわからないけど、のほほんと泳ぐ金魚が可愛いくて癒された。

最後に

昔ながらの和とモダンな感じを上手く融合させた光景はとても幻想的で綺麗だった。
あと浴衣で行ったらもっと楽しめたかな。来年あったら浴衣で行ってみたい。

おまけ

夜の日本橋はなんか好き。
